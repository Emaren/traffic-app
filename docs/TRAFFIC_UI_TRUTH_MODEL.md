# Traffic UI Truth Model

Traffic UI must not imply login from known IP/fingerprint matching.

## Display rules

- Use known identity chips for context only.
- Do not show text like "Active now" for stale or known-only identity signals.
- Fresh live movement may use a small live dot.
- Login/auth language belongs to AoE2WAR app/auth data, not Traffic identity mapping.

## Fixed behavior

Known player signals such as Jim remain visible in broad history, but they do not appear as audience-grade humans unless confirmed by independent behavior.
